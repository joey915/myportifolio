**Project Description**

The Personal Portfolio Website is a responsive and modern web application designed to showcase my skills, projects, and professional background. 
It includes multiple sections such as Home, About, Projects, and Contact, allowing visitors to learn more about me and explore my work. 
The website features a clean user interface, dark-themed design, and interactive elements to enhance user experience.

**Technologies Used**
HTML5 - Semantic structure and layout
CSS3 - Styling, animations, responsive design
JavaScript - Interactivity and DOM manipulation
Google Fonts - Typography
Git & GitHub - Version control
